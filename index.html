Fleet Management

The fleet management web application is designed to facilitate daily vehicle management and reduce maintenance costs. It allows users to view all vehicles in one place, and monitor their condition, costs, and location in real time. The application automatically generates reminders for upcoming services, technical inspections, and insurance expirations, ensuring that all key tasks are completed on time. With the ability to analyze data, users can make better decisions about fleet optimization and resource planning, thereby achieving greater business efficiency and productivity.

Record of Sport Equipment

The main task was to devise and design a system, or rather an application, for recording sports equipment. The goal was to build an application that would facilitate administrative and scientific tasks related to recording and monitoring sports equipment, as well as recording some other items related to a sports equipment store (such as the equipment itself, employees, jobs, tenders, costs, etc.). The application consists of several basic (independent) units that are upgraded with several smaller dependent parts. The system needs to be implemented in the simplest possible way so that it is accessible to everyone who wants to participate in documenting sports equipment.
